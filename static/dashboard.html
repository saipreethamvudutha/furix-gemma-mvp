How SIEM works

Upload security logs (or load the sample). The backend normalises them to ECS, fans each event through the rule / UEBA / ML detector lanes, accumulates per-entity risk, correlates multi-stage campaigns, scrubs PII, and asks Gemma for an incident report. You watch every step happen live.
