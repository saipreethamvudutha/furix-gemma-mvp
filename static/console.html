Routing & Cost — deterministic rules vs Gemma, per log

Runs every sample log through the full appliance and shows, for each one, which tier decided it. Most events are resolved by instant deterministic rules (regex + crosswalk + embeddings) — 0 ms, no model. Only events that genuinely need reasoning (or that no rule can map) fall through to a Gemma LLM call, with its real response time shown. The cost story: the model is the rare last resort, not the default.

⏱ First Gemma call may be slow (model cold-load). High-severity logs trigger the narrative agents (Remediation/Report) which call Gemma — those rows take longer; low-severity logs stay fully deterministic at ~0 ms.
